TodoClient

Angular Application Client

This is a sample application made using Angular for demonstaration purposes. Angular is used to consume data from an API that contains todo list items.
The application is hosted on an azure application service where the static files are ran in the server's wwwroot folder.

Github Respository: https://github.com/eddrichjanzzenang/TodoClient.git

TodoAPI

Basic Create, Read, Update, Delete

Outlined below is a table showing all possible endpoints for this simple todo list API

Github Respository: https://github.com/eddrichjanzzenang/TodoAPI.git
HTTP Method Route Type Action
GET <url>/api/TodoItems JSON List Returns a list of all todo items in the todo list.
GET <url>/api/TodoItems/filter?completed=true | false JSON List Returns completed items in the todo list.
GET <url>/api/TodoItems/<$id> JSON Returns one todo item given the its id.
POST <url>/api/TodoItems/<$id> JSON Creates a new todo item.
PUT <url>/api/TodoItems/<$id> JSON Updates an existing todo item given its id.
DELETE <url>/api/TodoItems/<$id> JSON Deletes a todo item given its id.
